# fuzzy-rubber-ducks

The Fuzzy Rubber Ducks app is a social media platform for developers to share their pets.

# Technologies Used

- Node.js
- Express
- HTML
- CSS
- Bootstrap
- Mongoose for MongoDB
- Passport for Oauth
- Third party API for breed information
